## 管理内容

> 此栏目是基于已建立的栏目的基础上，选中栏目进行内容的新增、编辑、删除管理。比如新闻管理、产品管理。。。

内容列表：

![内容列表](/images/content-1.png "内容列表")

***
新增/编辑：

![新增/编辑](/images/content-add.png "新增/编辑")

| 字段名称 | 说明 |
|  ----  | ----  |
| 栏目 | 该内容是属于哪一个栏目下的 |
| 标题 | 文章的标题 |
| 副标题 | 第二标题 |
| 描述 | 简要描述 |
| 预览图 | 文章预览图 |
| 状态 | 启用或禁用 |
| 排序 | 数字越小越靠前 |
| 内容 | 内容由一个富文本编辑器提供录入 |

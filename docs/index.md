# 欢迎使用yugeCMS!

yugeCMS是一个轻量级的可应用于官网的cms内容管理系统，分为后台管理系统和前台的官网页面，全部前后端分离。后端基于eggjs + mysql开发。
后台管理系统的前端基于antd、一款基于react的前端框架；前台官网页面基于umi框架，整合进eggjs进行服务端渲染。
